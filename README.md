# Bitcoin-Explorer
A bitcoin explorer with a feature where you can interact with like minded bitcoiners.
Created using ReactJS and API integration, showcasing real-time data about the Bitcoin blockchain. 
Developed user-friendly interface, search functionality along with user authentication and introduced comments system using Nodejs, Expressjs and MongoDB

It provide the real time data about the Bitcoin Blockchain. Precisely, On its main page it shows 10 blocks that has mined recently.
It has a search fuctionality where user can search any transaction using transaction hash and any block using blockhash or height of the block.

It also has some famous transactions of bitcoin blockchain
for example: "The first transaction"
"the bitcoin pizza day"
on clicking on them user can see details about it.

It has user authentication (used jwt for token based authentication when a user logs in, the server generate a jwt token with some time limit, which is then sent to the clinet) to access comment system which is powered by nodejs, expressjs, and mongodb allowing user to engage and interact with each other.

it uses a free api provided by blockstream to fetch realtime data.
